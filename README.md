操作提示：<br></br>
推荐屏幕分辨率：1920*1080 界面分辨率500*900， 小于此值会导致显示不完全。
在游戏界面-<br></br>
  请切换到英文输入<br></br>
  W向上移动 A向左移动 S向下移动 D向右移动<br></br>
  Esc退出游戏<br></br>
  H射击<br></br>
  P暂停游戏<br></br>
  右上角暂停/播放按钮可以打开会关闭游戏声音<br></br>
  <br></br>
